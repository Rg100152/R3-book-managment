# R3 Book - Book Management System

A beautiful, fully functional **Book Management System** built as a single HTML file using HTML5, CSS3 (Tailwind) and Vanilla JavaScript.

![R3 Book](https://img.shields.io/badge/Status-Production%20Ready-brightgreen)

## ✨ Live Demo

You can try it live by opening the `index.html` file in any modern browser.

## 📖 About

**R3 Book** is a modern, responsive, and elegant Library Management System designed for small libraries, personal collections, or educational institutions. It works completely offline using the browser's `localStorage`.

### Key Features

- **📊 Modern Dashboard** with real-time statistics
- **📚 Complete CRUD Operations** (Create, Read, Update, Delete books)
- **🔍 Powerful Search** by title, author, or genre
- **📖 Borrowed Books Tracking**
- **🎨 Beautiful UI** with professional design & smooth animations
- **📱 Fully Responsive** (Desktop + Mobile friendly)
- **💾 Persistent Storage** - Data stays even after closing browser
- **🌐 Hindi + English Support**

## 🛠️ Tech Stack

- **HTML5**
- **Tailwind CSS** (via CDN)
- **Vanilla JavaScript (ES6)**
- **Font Awesome Icons**
- **Google Fonts**
